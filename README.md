# LinkTracker  
**Автоматическое отслеживание обновлений по ссылкам с уведомлениями в Telegram**

---

## Описание

**LinkTracker** — это система для мониторинга изменений по ссылкам.  
Когда в отслеживаемых источниках появляются обновления, вы получаете уведомления в Telegram.

Поддерживаемые источники:
- Stack Overflow — новые ответы и активности в топиках
- GitHub — коммиты, релизы, активности в репозиториях
- 



Приложение разделено на два независимо работающих сервиса:

- **Telegram-бот** — интерфейс для управления ссылками  
- **Scrapper-backend** — сервис, который регулярно проверяет ссылки и отправляет обнаруженные обновления боту

## 🔧 Переменные окружения

### 🟣 Telegram Bot

| Переменная        | Тип     | Обязательная | Описание |
|-------------------|---------|--------------|----------|
| `TELEGRAM_TOKEN`  | String  | ✔️           | Токен Telegram-бота, полученный от @BotFather |
| `SCRAPPER_URL`    | URL     | ✔️           | URL backend-сервиса Scrapper для отправки запросов |

---

### ⚫ Scrapper Backend

| Переменная        | Тип     | Обязательная | Описание |
|-------------------|---------|--------------|----------|
| `DB_URL`          | String  | ✔️           | JDBC URL базы данных PostgreSQL |
| `DB_USER`         | String  | ✔️           | Имя пользователя базы данных |
| `DB_PASSWORD`     | String  | ✔️           | Пароль пользователя БД |
| `BOT_URL`         | URL     | ✔️           | URL Telegram-бота для отправки уведомлений |

## 🛠 Используемые технологии

Проект построен на современном технологическом стеке, который обеспечивает надёжность, масштабируемость и лёгкость сопровождения.

### 🔹 Backend / Scrapper

| Технология        | Роль в проекте |
|-------------------|----------------|
| **Java 17**       | Основной язык разработки |
| **Spring Boot**   | Фреймворк, для быстрого  старта разработки сервиса |
| **Spring Web**    | Реализация REST-взаимодействия между сервисами |
| **PostgreSQL**    | База данных |
| **Spring Data**   | Работа с базой данных |
| **Hibernate**     | ORM |
| **Liquibase**     | Миграции и версионирование БД |
| **SLF4J**         | Централизованное логирование |
